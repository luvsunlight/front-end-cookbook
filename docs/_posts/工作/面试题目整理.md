## 美团笔试笔试(2020-3-11)

> 现在有一个数组不知道其内容，只知道长度为n，且每一个数字均可以在l和r之间（包含边界），且数组综合可以被k整除，求问这样的数组存在多少种可能

当时想的是用分治做，分治该怎么想想了好久，后来一想直接用回溯，非常非常简单，虽然效率可能慢了点

```
let n = 2,
    l = 1,
    r = 3,
    k = 3

let plans = 0

function traverse(startIndex = 0, total = 0) {
    if (startIndex < n) {
        for (let i = l; i <= r; i++) {
            traverse(startIndex + 1, total + i)
        }
    } else {
        if (total % k === 0) {
            plans += 1
        }
    }
}

traverse()
console.log(plans)
```
